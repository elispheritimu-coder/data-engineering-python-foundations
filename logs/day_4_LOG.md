# Day 4 – Loops & Repetition

## What I Did
- Practiced `while` loops using a counter
- Ran scripts using correct folder paths
- Built a loop that retries input until valid
- Learned how `break` exits a loop
- Debugged `break outside loop` using indentation fixes and file cleanup

## What I Learned
- `while` repeats as long as a condition is True
- `break` only works inside loops
- Indentation controls program structure in Python
- File names and paths must match exactly (hyphen vs underscore matters)
- If a script shows no output, it may be the wrong file being executed

## Challenges
- `break` outside loop error due to indentation / hidden formatting
- Confusion from accidentally creating duplicate files with similar names

## Confidence Level (1–10)
- 7/10

## Notes
- Use `pwd` + `ls` before running scripts
- Prefer tab-complete to avoid filename mistakes
