# Day 8 – Dictionary Audit & Reporting

## What I Did
- Used dictionaries with `.get()` to access keys safely
- Practiced updating dictionary values with `user["key"] = value`
- Iterated through multiple dictionaries in a list
- Learned and used `.items()` for key-value pair access
- Built a user audit system that:
  - Flagged missing emails, inactive users, missing countries
  - Displayed clean audit reports for each user
  - Counted total users, missing emails, inactive accounts, and more

## What I Learned
- How to safely access and modify dictionary data
- How to use nested loops to check data conditions
- How to track summary counts across loops
- The value of clean, readable audit reports in real-world roles

## Challenges
- Indentation issues that caused loops to skip
- Forgetting to check for `None` or empty strings properly
- Accidentally only printing last user report due to placement errors
